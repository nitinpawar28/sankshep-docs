# Feedback

Sankshep's product source is **private** and it does not accept external pull requests — but bug reports
and feature suggestions are genuinely welcome and read.

## Report a bug or suggest a feature

**→ [Open an issue on `sankshep-docs`](https://github.com/nitinpawar28/sankshep-docs/issues/new/choose)**

This public repository (the one hosting these docs) is the feedback channel. Please include:

- **For bugs:** what you ran, what you expected, what happened, and your OS + client (Copilot / Claude
  Code / Cursor…). A minimal repro helps a lot.
- **For suggestions:** the problem you're trying to solve and why it matters — motivation over
  implementation.

Suggestions are considered even though implementation stays in-house.

## Docs issues

Found a mistake or a gap in this documentation? The same [issue
tracker](https://github.com/nitinpawar28/sankshep-docs/issues) works — or use the "edit" pencil on any
page to point at the source.
